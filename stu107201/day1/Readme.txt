软件名称：用户登陆
开发者及版权所有：lzpython
应用范围：内部使用
功能：根据已经有用户账号密码文件，当用户密码输入正确时提示登陆成功，而当用户密码输错三次锁定该账号并不允许再次登陆。
使用方法：
    1.新建用户密码文本文件user，按行录入正确的用户账号和密码，格式如用户账号：密码。
    2.运行yhdl.py，提示“请输入用户名：”，此时在界面上输入用户名（user中保存的用户名），敲回车。 
    3.提示“请输入密码：”，当输入正确的密码敲回车时，提示“欢迎登录系统”；当输入错误密码时，系统再次提示“请输入密码：”        继续输入密码，当连续三次输入密码错误时，系统提示“（用户名） 你的密码输入超过三,锁定帐号,退出”，同时将该用户名保存到       lock文件中。当再次运行程序输入刚锁定的用户名时，系统提示“（用户名） 已经被锁定,请联系管理员.退出!”。



输入名称：多级菜单
开发者及版权所有：lzpython
应用范围：内部使用
功能：根据用户选择，显示省级、区县级、乡镇级名称。
使用方法：
     说明：该程序只为实现层级展现逻辑，只实现了选“北京”，选“湖北”分支暂时未实现。
     1.运行djcd.py，显示如下：1.北京
                              2.湖北
                              >>请选择:
     2.输入操作如下（包括一级一级选择进入和回退）
         1.北京
         2.湖北
         >>请选择:1
          --1.东城
          --2.昌平
         >>请再次选择:1
         ----1.安定门街道
         ----2.建国门街道
         >>请再次选择:b
         --1.东城
         --2.昌平
         >>请再次选择:b
         1.北京
         2.湖北
         >>请选择:1
         --1.东城
         --2.昌平
         >>请再次选择:2
         ----1.小汤山镇
         ----2.阳坊镇
         >>请再次选择:b
         --1.东城
         --2.昌平
         >>请再次选择:
     3.当进入任何一层菜单时，用户输入q，程序立即退出。如下：
         1.北京
         2.湖北
         >>请选择:1
         --1.东城
         --2.昌平
         >>请再次选择:1
         ----1.安定门街道
         ----2.建国门街道
         >>请再次选择:q

          Process finished with exit code 0
